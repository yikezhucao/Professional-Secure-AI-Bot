ls 是 Linux 中最基础的命令之一，用于查看目录中的文件和子目录。你可以简单输入 ls 来列出当前目录下的内容，加入 -l 参数可以显示详细信息，如权限、所有者、大小和修改时间，而 -a 参数则会显示所有文件，包括以点开头的隐藏文件。这个命令常用于快速查看文件结构和状态。

cd 用于在 Linux 中切换当前工作目录，是文件导航的核心命令之一。例如，cd /var/log 可进入系统日志目录，而使用 cd .. 可以返回上一级目录，输入 cd 或 cd ~ 则会回到当前用户的主目录。掌握 cd 能帮助你在终端中快速移动，提升工作效率。

pwd（Print Working Directory）用于显示当前所在的工作目录的完整路径。当你在终端中多次使用 cd 命令切换目录后，pwd 可以帮助你确认你目前所处的位置，这在编写脚本或查找文件路径时尤其重要。

cp 用于复制文件或目录，是管理文件的关键工具。你可以通过 cp file.txt backup/ 来将文件复制到备份目录，如果需要复制整个目录，可以加上 -r 参数进行递归复制。使用 -i 参数可以在覆盖文件前提示确认，从而避免误操作。

mv 既可以用于移动文件，也可以用于重命名文件。例如，mv file.txt newname.txt 可以为文件改名，mv file.txt /tmp/ 则可以将文件移动到临时目录。它不像 cp 那样保留源文件，而是直接对文件或目录进行“搬家”操作，省空间也更高效。

rm 是 Linux 中用于删除文件或目录的命令，使用时需小心谨慎。一个简单的 rm file.txt 会删除文件，而 rm -r myfolder 则会递归删除整个目录。加上 -f 参数可以强制删除而不提示，但更推荐加 -i 进行交互式确认，防止误删重要文件。

touch 主要用于创建一个新的空文件，或者更新已有文件的访问和修改时间戳。输入 touch newfile.txt 就能快速建立一个空文本文件，适用于批量文件创建、自动化脚本初始化等场景，是系统管理员的常用工具之一。

cat 可用来查看文件的内容，尤其适用于快速查看小型文本文件。例如 cat /etc/hosts 可以显示本地主机配置文件的内容。它还能用于多个文件拼接输出，是日志查看、配置检查中最常见的基础命令之一。

grep 是强大的文本搜索工具，用于在文件或命令输出中查找匹配的关键词。比如 grep "error" /var/log/syslog 可以帮你从日志中快速找出包含 “error” 的行。通过配合 -i 忽略大小写、-r 递归搜索目录等参数，grep 成为日志分析和排查故障时不可或缺的利器。

chmod 用于修改文件或目录的权限，是 Linux 权限控制体系的关键命令。你可以使用类似 chmod 755 script.sh 的命令设置脚本为可执行，其中数字权限表示用户、组、其他用户的读写执行权限。理解权限设置对于保护系统安全、控制文件访问具有重要意义。

mkdir 是用于创建新目录的命令，通过输入 mkdir myfolder 可以快速建立一个名为 myfolder 的新文件夹。若要一次性创建多层目录，可使用 mkdir -p parent/child/grandchild，这在自动化脚本中尤其有用，能避免因上级目录不存在而报错。

rmdir 是用于删除空目录的命令，和 rm -r 相比，它更加安全，但功能有限。执行 rmdir testfolder 只能删除完全为空的目录，否则会提示错误。适合用于清理临时创建的空目录结构。

echo 是一个非常灵活的命令，可以用来在终端输出文本内容，也常用于脚本中生成变量内容或写入文件。例如 echo "Hello, World!" 会输出一行文本，而 echo $HOME 则会打印当前用户的主目录路径。

man 命令用于查看其他命令的使用手册，是学习 Linux 的好帮手。输入 man ls 就可以查看 ls 命令的所有用法、参数说明和示例。配合 /关键字 搜索和 q 退出，可以高效地浏览命令说明文档。

head 命令用于查看文件的前几行内容，默认显示前 10 行，例如 head logfile.txt。你也可以使用 -n 参数自定义行数，如 head -n 20 logfile.txt，非常适合快速查看大文件的起始结构。

tail 命令与 head 相对，用于查看文件的最后几行内容，是日志监控的常用工具。输入 tail -f /var/log/syslog 可以实时监控系统日志的最新动态，适用于故障排查和后台服务监控。

df 是查看磁盘空间使用情况的命令，通过 df -h 可以以人类可读的格式（如 GB、MB）展示每个挂载点的总容量、已用空间和剩余空间，是系统资源检查的常用工具。

du 用于查看文件或目录的实际磁盘使用量。输入 du -sh /home/user 可以快速查看某个目录的整体空间占用，-s 表示只显示汇总，-h 则是人类可读格式。适合定位磁盘空间占用大的目录。

ps 命令用于查看当前运行的进程状态，通过 ps aux 可以列出所有进程及其资源使用情况，包括 PID、CPU、内存等。结合 grep 可以查找特定程序进程，如 ps aux | grep nginx。

kill 用于向进程发送信号，最常见的是终止某个进程。先用 ps 或 top 查到进程 PID，再使用 kill 1234 来结束它。如果进程无法正常结束，可以使用 kill -9 1234 强制终止。需要注意安全，避免误杀关键服务进程。
